# HTML语法结构及规范

## 1. 概念
- 是HTML种超文本标记语言;

## 2. HTML注释

```
<!--注释内容-->
```
注释的作用：代码说明;

## 3. 语法结构
- 符号
尖括号<>;  
所有的HTML语言都必须括在其中;
- 标签格式
双标签：成对出现 <标签名></标签名> <titile>HTML语言</title>
单标签：<br/>只存在开始标签，不需要结束;
双标签:通常用于包含内容的元素,单标签表示无内容的元素。

## 4. 结构  
3个组成部分：标签 属性 值  
a.标签可以用于属性，属性和值都写在开始标签的尖括号中；  
b.标签和属性之间用空格隔开，多个属性之间也用空格隔开；  
c.如果属性有对应的值，则属性和值之间事等式关系，且值需要在英文引号内。    
语法结构：<标签名 属性="值" 属性="值">...</标签名 >

## 5. 规范
- 标签名和属性名都使用小写;
- 属性的值用双引号括起来;
- 标签的使用符号嵌套规则;

## 6. 编写网页的流程
首先，找参考；
其次，准备素材/建站；
再次，添加内容元素；
最后，标签结构化。

## 7. 基本结构

```
<!DOCTYPE html><!--html文档的文档类型声明-->
<html><!--html文档的开始-->
	<head><!--主要用来说明html文档的相关属性的或设值-->
		<meta charset="UTF-8"><!--指定网页的编码-->
		<title></title><!--设定网页的标题-->
	</head>
	<body><!--html文档的主体区域，对应浏览器的呈现内容-->
	</body>
</html><!--html文档的结束-->
```

# Markdown基础语法笔记 
### 1. 字体形式（实例）
- 倾斜：
*这是倾斜文字*  
文字两边分别用一个 * 或 - 将其包围
- 加粗：**这是加粗文字**  
文字两边分别用两个 * 将其包围  
- 倾斜加粗： ***这是倾斜加粗字体***  
文字两边分别用三个 * 将其包围  
- 下划线：~~这是删除线~~  
文字两边分别用两个 ~ 将其包围  
Tip：符号与文字之间无需空格  

### 2. 分级标题    

```
# 这是一级标题
## 这是二级标题  
### 这是三级标题
```

方法一：在标题文字前使用 # ，几个 # 代表几级标题。  
Tip：符号与标题文字之间需要一个空格    

### 3. 超链接  
（1）图片链接  

```
！[图片描述](图片路径)  
``` 

（2）自动链接  
<链接地址>  

### 5. 代码块  
多行代码块与语法高亮：在需要高亮的代码块的前一行及后一行使用三个单反引号“`”包裹，就可以了。

### 6. 引用  
（1）基本用法  
> 这是引用部分  
方法：在被引用的文本前加上 > 符号。 
```
> 这里是引用文字
```
Tip：被引用文本与符号间存在一个空格，如若没有空格会产生空白引用。  
（2）嵌套用法  
```
>>>引用1  
>>引用2  
>引用3  
```

### 7. 列表  
Tip：在使用列表时，只要是数字后面加上英文的点，就会无意间产生列表，比如2019.12.11这时候想表达是日期，有些软件把它误认为是列表。解决方式是在每个点前面加上 \ 即可。
（1）无序列表  
方法：在文字前使用一个* / + / - Tip:符号与文字之间存在一个空格，起缩进作用。  
实例：
```
* 这是无序列表  
- 这是无序列表  
+ 这是无序列表  
```
* 这是无序列表  
- 这是无序列表  
+ 这是无序列表  
（2）有序列表  
方法：使用数字和一个英文句点表示有序列表。  
Tip：英文句点与文字间存在一个空格，起到缩进作用。  
实例：
```
1. 这是有序列表  
2. 这是有序列表  
3. 这是有序列表  
```
1. 这是有序列表  
2. 这是有序列表  
3. 这是有序列表
（3）列表和其他要素混合使用   
Tips： 
1. 加粗效果不能直接用于列表标题里面，但是可以嵌套在列表里面混合使用。
2. 列表中包含代码块（前面加两个tab或者8个空格，并且需要空一行，否则不显示）
